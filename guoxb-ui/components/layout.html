    # 样式

    .grid-content {
  min-height: 36px;
}
.bg-purple-dark {
  background: #99a9bf;
}
.bg-purple {
  background: #d3dce6;
}
.bg-purple-light {
  background: #e5e9f2;
}
.g-row {
  margin-bottom: 10px;
}

    # 配置选项

    # g-row 属性

    参数 说明 类型 可选值 默认值
    gutter 栅栏间隔 number - 0

    # g-col 属性

    参数 说明 类型 可选值 默认值
    span 栅格占据的列数 number - 24
    offset 栅格左侧的间隔格数 number - 0
