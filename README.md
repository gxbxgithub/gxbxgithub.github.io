# 导读

个人练习 vue组件开发 的 Demo，模仿 [Element组件](https://element.eleme.io/#/zh-CN)

### 安装

``` sh
npm install guoxb-ui -S
```

### 引入

``` js
//main.js
import GuoxbUI from 'guoxb-ui'
import 'guoxb-ui/dist/guoxb-ui.css'
Vue.use(GuoxbUI)
```

### 使用

``` html
<g-button></g-button>
```
